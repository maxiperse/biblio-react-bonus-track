# BiblioApp - Proyecto React SPA

**Estudiante:** Maximiliano Robles
**Asignatura:** Programación Front End 

## Descripción
Catálogo digital interactivo desarrollado en React. Permite listar libros precargados, filtrar por estado (Disponible, Prestado, Reservado) y buscar por título.

## Instalación y ejecución
1. Clonar el repositorio.
2. Ejecutar `npm install` en la terminal.
3. Ejecutar `npm run dev` para levantar el servidor local.
